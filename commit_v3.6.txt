feat(core): launch V3.6.0 Command Center SPA & SYNAPSE Copilot 🧬

This massive update fundamentally rewrites Project Helix's frontend architecture, migrating from a static Streamlit dashboard to a decoupled, high-octane Single Page Application (SPA) powered by FastAPI and Vanilla JS/Tailwind.

🚀 MAJOR FEATURES:
- [SYNAPSE Copilot]: Introduced a built-in GLM-5 conversational UI. Added the ability to mount local Vault intelligence packages (.md) directly into the LLM's context window for real-time analysis.
- [Secure Vault GUI]: Built an in-browser file explorer and Markdown decryptor. Features explosive GSAP `clip-path` entry transitions, dynamic `ALL/ARXIV/GITHUB` filtering, and active-file visual tracking.
- [Targeted Hunting]: Upgraded the GitHuber engine and API to accept dynamic search queries. If left blank, it smartly falls back to the default trending algorithm.

🧠 UX & STATE MANAGEMENT:
- [Shared Memory Architecture]: Implemented `tabMemory` caching. Arxiv and Github tabs now share a UI deck but perfectly preserve their individual inputs and terminal outputs during navigation.
- [Ghost Navigation]: Designed a cyberpunk-inspired "Ghost Button" sidebar with neon-green slide-in hover states and active indicators.
- [Purity Protocol]: Automated screen-clearing and dynamic placeholder switching when navigating between different engine strands.
- [Scroll Liberation]: Patched Lenis smooth-scrolling conflicts (`data-lenis-prevent`) to allow seamless independent scrolling within the Vault and Chat interfaces.

🛠️ BACKEND:
- Replaced Streamlit with a FastAPI backend (`api.py`).
- Added `/api/chat` route for GLM-5 streaming/blocking generation.
- Updated `/api/github/ignite` to parse and transmit custom user payloads to the backend scraper.

"We didn't just update the UI; we gave it a brain."
